VMware虚拟机安装windows 1.0

设置虚拟机

选择自定义，然后选择其他里面的MS-DOS

设置虚拟机名称，以及存放的路径

设置内存，不用跟图上一样默认即可。

然后一直下一步下一步，将要完成时点击自定义硬件

添加软盘驱动器

选择DOS7.1目录下的DOS71_1.IMG

安装MS-DOS7.10

然后开机，根据提示操作即可(英文看不懂的可以自行百度👀)，到如下这个界面时选择挂载DOS7.1目录下的DOS71_2.IMG

继续根据提示下一步，安装成功后会让你重启。重启开机的时候会出现如下界面

安装Windows 1.0

然后开机后是一个命令行，如下所示。软盘驱动器设置使用Windows 1.0的DISK1.IMG，这时在命令行输入A:,随后在命令行输入setup开始安装。

遇到如图所示更换为DISK2，根据提示以此类推

到出现下面界面时按N即可完成安装输入win回车就能进入系统，按Y会随后再安装一些其他的东西，我这里选择的是Y

安装完成输入win然后回车

进入系统如下所示

记得要将软盘驱动器断开连接，否则重启后开机会出现如下错误

VMware虚拟机安装windows 2.0

大致与前面相似，按回车

接下来Windows将让我们选择使用的电脑和显示器等。选择如下。选择No Change回车

点键盘上的i键

根据指示依次切换软盘文件

按F完成安装

开机后如下所示

VMware虚拟机安装windows 3.1

有两种安装方式一种是跟上面一样的用软盘安装，还有一种是使用文件安装的方式(可能叫法不太正规😂)。Windows 3.1简体中文版的没找到那个软盘镜像，所以采用第二种方式安装。

安装好MS-DOS7.1后关机，然后选择磁盘映射把那个Windows 3.1文件夹复制到里面,然后改个名字改成install当然也可以不用改

断开连接后开机，使用cd命令进入install目录然后输出setup开始进行安装

注意内存不能超过256MB，要不然会报如下错误

当然也不是没有解决方法，最简单的就算内存设置成小于等于256MB，还有一个方法就是修改SYSTEM.INI文件在末尾添加一行

链接🔗

VMware虚拟机安装windows 3.2

windows 3.2采用软盘安装的方式(稍微有点麻烦)，需要不停的切换软盘进行安装。建议采用第一种方法来，在这里我只是想演示一下不同的安装方式。主要步骤跟安装Windows 1.0类似,这里不再详细说明按照指示进行安装就🆗

Windows 3.2跟Windows 3.1类似看起来几乎一模一样

安装软件

首先安装一下VMware tools然后把win98软件复制进去，文件夹内是声卡驱动安装包以及opera和7.zip的安装包等

安装好后虚拟机就有声音了并且可以使用浏览器，压缩解压软件。

这里分享一个网站，有很多旧系统软件的安装包可以自行下载安装并体验🔗链接这个网站并非只有Windows平台的软件，还有Android、Linux、MAC

总结

安装这些没什么意义，不值得去浪费时间来安装这些老古董系统。上述这些仅仅只是一部分，还有部分上面并未提到。但是系统以及系统的安装方法都是大同小异的！！！建议不要把时间花费到这个上面，除非你真的是特别的无聊。
